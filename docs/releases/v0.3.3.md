# v0.3.3 — shipped-agent and contract accuracy

A patch release of four surgical accuracy fixes to shipped agents and the contract — each corrects existing wording/ordering, no new mechanisms. These improve the agents consumers run; no installer changes.

## Fixes

- **#260 — oversight-evaluator fail-open wording.** The Inputs section stated unconditionally that a present second-review file with `verdict: skipped` is "valid", contradicting the Phase-1 rule that `skipped` on a **MEDIUM+** validated-tier step is a COMPLIANCE FAIL. A reader of the summary could accept a skipped cross-vendor review on exactly the steps where it's mandatory. The Inputs sentence now states the tier condition (valid only below MEDIUM; MEDIUM+ → COMPLIANCE FAIL).

- **#263 — dep-mapper step ordering.** `Step 3.5 — Self-detect coverage gaps` physically appeared *after* `Step 4 — Classify the blast radius`, so an agent reading top-to-bottom classified the blast radius before running the self-detect logic that sets `Data confidence: LOW` — the signal Step 4 depends on. Step 3.5 now precedes Step 4.

- **#264 — oversight-orchestrator vs AGENTS.md disclosure mismatch.** The orchestrator hardcoded a PR-disclosure table row and a merge-gate sentence that disagreed with the canonical "Universal AI disclosure requirement" in `docs/AGENTS.md` (it asserted human approval is *always* required, where AGENTS.md allows a LOW-risk non-protected change to be overseer-approved). Reconciled to AGENTS.md's wording (AGENTS.md unchanged, as the source of truth).

- **#266 — contract §6a event catalog completeness.** §6a declares its event catalog "canonical — every line written to the log uses one of these event types," but the system emits two it didn't list: `human-authorization` (oversight-evaluator) and `hos-prune` (hos_install.sh). Both added with their real meaning/emitter/key-fields, so the catalog is authoritative again.

## Upgrading
No action needed for installed projects; the next framework upgrade picks these up. Consumers on the v0.3.x layered model receive the corrected agents in their CORE regions.
