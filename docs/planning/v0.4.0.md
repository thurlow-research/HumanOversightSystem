**Theme:** Make the worker/overseer loop truly autonomous — trustworthy identity, reliable PR protocol, disciplined pipeline, and verifiable quality.  
**Status:** ✅ **Shipped** 2026-06-20T17:11Z — https://github.com/thurlow-research/HumanOversightSystem/releases/tag/v0.4.0  
**Milestone:** [v0.4.0 — Autonomous Worker](https://github.com/thurlow-research/HumanOversightSystem/milestone/1)  
**Gate:** #557 (pre-release review gate runs last)

---

## Theme rationale

v0.3.x established the governance framework. v0.4.0 makes it run unattended. The core question: can the worker and overseer operate over a full day without a human holding their hand? Three gaps blocked that: the identity model used personal credentials (not auditable), the PR review protocol missed feedback from the overseer, and the pipeline discipline was honor-system only. This release closes all three.
